# Button

Versatile action button supporting intents, sizes, loading, and full-width.

## Usage
```tsx
import { Button } from '@kasomaibrahim/tw-component-library';
<Button intent="primary">Save</Button>
<Button intent="secondary" tw="rounded-full">Cancel</Button>
<Button as="a" href="#" intent="ghost">As link</Button>
```
